# template-npm-ts

> soa template for npm module typescript

## Usage

Install [SAO](https://github.com/egoist/sao) first.

```bash
$ yarn global add sao
# or
$ npm i -g sao
```

### From git

```bash
$ sao zcong1993/template-npm-ts my-project
```
